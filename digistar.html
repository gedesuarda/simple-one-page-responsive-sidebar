Mekanisme Program Digistar Q3 2020

Dear Kakaks2 yth...

Bersama ini disampaikan bahwa Program DigiSTAR Q3 2020 sudah resmi berjalan per-1 Juli sd 30 September 2020. Berikut beberapa catatan (detail program terlampir) :

1. Bintang yang ter-create di Program DigiSTAR sebelumnya akan terakumulasi dengan Bintang DigISTAR Q3 2020 (tidak hangus)
2. Redemption tetap akan dibuka setiap hari*, dengan batasan maks 1jt/hari
3. Terdapat beberapa perubahan parameter & Bintang trx dari Program DigiSTAR sebelumnya.
4. untuk saat ini BINTANG Q3 masih belum muncul di Aplikasi Digipos, namun demikian seluruh trx di Q3 tetap diperhitungkan sesuai mekanisme Program DigiSTAR Q3, dan akan dimunculkan kemudian.
5. Informasi DigiSTAR Q3 2020 akan kami broadcast ke outlet melalui SMS 1818 dan via Instagram @digistar.telkomsel

Demikian yang dapat disampaikan.
Terimakasih atas perhatian dan supportnya.
